H‑Talk (현대차 스타일) — 무설치 실행 가이드
================================================
구성:
- index.html : CDN React를 사용한 실행 진입점
- app.js     : 앱 로직 (자동 대화 생성, 중복 방지, 요일 규칙, 메인방 20개/일)
- styles.css : 기본 스타일

실행 방법:
1) 폴더를 압축 해제한 뒤, index.html을 더블클릭하면 브라우저에서 바로 실행됩니다.
   (인터넷 연결이 필요합니다 — React CDN 로딩용)
2) 메인방(r1)에 하루 20개의 자동 대화가 생성되며,
   - 나(백인헌)는 월/목/금/토에만 자동 참여
   - 다른 팀원은 월~토 자동 참여
   - 시간대는 오전(9–11)/오후(12–17)/저녁(18–21) 중 랜덤
   - 문장 중복 방지(고정 문구 + 조합형 문장)

메시지 입력:
- 하단 입력창에 글을 쓰고 Enter(또는 전송 버튼)를 누르면 내가 보낸 말풍선이 오른쪽에 표시됩니다.
- 20초 후 무작위 팀원이 자동으로 답장을 보냅니다.

초기화:
- 사용 이력(마지막 접속 시각, 사용한 문장 풀)은 localStorage에 저장됩니다.
- 브라우저에서 개발자 도구 → Application → Local Storage에서 `htalk_*` 키를 삭제하면 초기 상태로 돌아갑니다.
